# 欲夜低吟 - 18+ 情侣私密真心话大冒险（纯前端奢华版）

一款专为情侣（支持 **男女 MF**、**男男 MM**、**女女 FF** 等各种性别与伴侣组合）设计的 **R18+ 级私密互动真心话大冒险**。

本项目采用 **100% 纯前端架构**，完全运行在浏览器中，**不收集任何用户数据**。所有配置信息、自定义专属题库、游戏状态全部加密存储在玩家的本地浏览器缓存（LocalStorage）中，保障绝对的隐私与私密安全性。

## ✦ 核心亮点特色 ✦

1. **多元情侣智能适配**
   - 可输入两位玩家的昵称，并各自选择人称代词（男：**他** / 女：**她** / 其他：**TA**）。
   - 题库模板内置**智能语法引擎**，抽中卡片后人称代词、物理接触方向和动作会根据双方性别**完美契合、动态翻译**。支持同性情侣和异性情侣！

2. **4 级渐进欲望温控**
   - 🟢 **Level 1: 微醺温存 (Warm Up)** —— 甜蜜纯爱互动，温和的真心话，轻微的身体接触。
   - 🟡 **Level 2: 暧昧调情 (Flirting)** —— 带有暗示性的话题、挑逗性肢体接触与亲吻。
   - 🟠 **Level 3: 烈火激情 (Passion)** —— 深入敏感带的爱抚、衣物脱除、半蒙眼/轻度束缚挑战。
   - 🔴 **Level 4: 狂热禁忌 (R18+ Explicit)** —— 极度私密、高尺度探索与性幻想，包含大尺度成人姿势与深度交互。

3. **极奢深色玻璃态视觉 (Glassmorphism & Neon)**
   - **磨砂玻璃卡片**：配有细腻流光边框、阴影和背景模糊滤镜。
   - **动态粒子背景**：利用 HTML5 Canvas 渲染的夜空浪漫浮光，优雅且流畅（60 FPS）。
   - **3D 翻牌动画**：丝滑的 CSS3 3D 旋转卡片动效，增加仪式的刺激感。
   - **纯图标网站**：严格禁用任何 Emoji 表情，全面采用高大上的 **Lucide 矢量图标** 网页专用图标，逼格拉满。

4. **Web Audio API 声音合成引擎**
   - 无需加载庞大的外部音频 MP3 文件，利用浏览器内置音频发生器**动态合成**点击、发牌、翻转、成功、警告/低沉惩罚音效，带来优雅听觉反馈。

5. **本地专属题库抽屉 (Drawer)**
   - 设有侧拉式悬浮“自定义题目”管理器，支持随时录入两人专属的敏感词、悄悄话或大尺度命令，随时删除或添加，同样百分百保存在本地。

## 🌐 快速部署到 GitHub Pages (情侣联机)

由于本项目是 **纯静态前端网页**，最完美的部署方式就是使用免费的 **GitHub Pages**。部署后，生成专属的独立网址，您和伴侣只需通过手机访问该网址即可异地或面对面游玩！

### 部署步骤：

1. **注册并登录 GitHub**
   - 打开 [GitHub 官网](https://github.com/) 注册或登录您的账号。

2. **新建仓库 (Repository)**
   - 点击右上角的 `+` 号，选择 **New repository**。
   - **Repository name**（仓库名称）：输入一个名字，比如 `love-game`。
   - **Public/Private**：请务必选择 **Public**（公开，GitHub Pages 必须公开仓库才能免费部署）。
   - 其他选项（如 Add a README.md）保持默认不勾选，点击底部的 **Create repository** 按钮。

3. **上传网页文件**
   - 在新建好的仓库页面中，点击中间蓝色小字的 **uploading an existing file**（上传已有文件）。
   - 将本地的 **4 个文件**（`index.html`、`styles.css`、`game.js`、`database.js`）拖入网页的上传区域。
   - 等待文件上传完毕，在下方的 **Commit changes** 输入框中填写“Initial commit”（或保持默认），点击绿色按钮 **Commit changes**。

4. **开启 GitHub Pages 部署**
   - 点击该仓库顶部的 **Settings**（设置）选项卡。
   - 在左侧侧边栏中找到 **Code and automation** 区域，点击 **Pages**。
   - 在 **Build and deployment** 下的 **Source** 选择 `Deploy from a branch`。
   - 在 **Branch** 栏下拉菜单中，将 `None` 改为 `main`（或 `master`），右侧的目录保持为 `/ (root)`，然后点击右侧的 **Save** 按钮。

5. **获取游戏网址**
   - 等待 1~2 分钟，刷新当前页面。
   - 在页面顶部会出现一个醒目的横幅，写着：**"Your site is live at..."**，后面紧跟的链接就是属于你们的专属私密游戏网址！
   - 例如：`https://你的用户名.github.io/love-game/`
   - 将这个链接发给你的伴侣，使用手机浏览器（推荐 Safari、Chrome、夸克等）打开即可开始激爽游玩！
