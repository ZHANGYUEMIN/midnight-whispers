## 🌐 部署到 GitHub Pages / Deploy to GitHub Pages

### 🇨🇳 部署步骤
由于本项目是 **纯静态前端网页**，最完美的部署方式就是使用免费的 **GitHub Pages**：

1. **登录 GitHub**：打开 [GitHub 官网](https://github.com/) 登录您的账号。
2. **新建仓库**：
   - 点击右上角 `+`，选择 **New repository**。
   - **Repository name**（仓库名称）：输入 **`midnight-whispers`**。
   - 请务必选择 **Public** 选项（公开仓库才能免费部署 GitHub Pages）。
   - 点击底部的 **Create repository**。
3. **上传网页文件**：
   - 在新建好的仓库页面中，点击 **uploading an existing file**。
   - 将本地的 **5 个文件**（`index.html`、`styles.css`、`game.js`、`database.js`、`README.md`）拖拽上传。
   - 点击绿色按钮 **Commit changes**。
4. **开启 Pages 服务**：
   - 点击仓库顶部的 **Settings** 选项卡。
   - 在左侧边栏中找到 **Pages** 选项。
   - 在 **Branch** 栏下拉菜单中，将 `None` 改为 `main`（或 `master`），然后点击右侧的 **Save**。
5. **获取游戏网址**：
   - 刷新页面，顶部会出现横幅：**"Your site is live at [专属链接]"**。
   - 将这个链接发给伴侣，用手机浏览器打开即可开始激爽游玩！
